# macos

**API endpoint**

POST `/fregata/v1/macos`

**input params:**

```json
{
    "title": "title",
    "subtitle": "subtitle",
    "message": "message",
    "sound": "sound"
}
```

Sounds list located at: `/System/Library/Sounds`

such as:

- Basso
- Blow
- Bottle
- Frog
- Funk
- Glass
- Hero
- Morse
- Ping
- Pop
- Purr
- Sosumi
- Submarine
- Tink
